# Empresa X — Site / Proposta Comercial

Site institucional e proposta comercial da **Empresa X**, agência de
**marketing de conteúdo, identidade visual e gestão de redes sociais**.

Site estático (HTML + CSS + JavaScript), sem build. Layout em **tela dividida**
com tema **Claymorphism** (azul/branco), totalmente responsivo, com WhatsApp
integrado e um **configurador "Monte seu plano"** que gera a estimativa e envia
o resumo pré-preenchido para o WhatsApp.

## Estrutura

```
agencia/
├── site/
│   ├── index.html
│   ├── css/styles.css
│   └── js/main.js
├── CONTEXTO.md                        # contexto do projeto
├── ATUALIZACOES.md                    # histórico de mudanças
└── Proposta_Comercial_Empresa_X.docx  # documento original
```

## Como rodar

Abra `site/index.html` no navegador (duplo clique) ou sirva a pasta `site/`
com qualquer servidor estático.

## Publicar com GitHub Pages

Em **Settings → Pages**, selecione a branch `main` e a pasta `/root`, depois
acesse `.../site/`. Alternativamente, mova o conteúdo de `site/` para a raiz.
